Sobre mi

Durante este tiempo, he adquirido un conocimiento sólido en programación, familiarizándome con lenguajes como Java, C#, JavaScript, y también con tecnologías web como HTML y CSS. Me siento muy cómodo utilizando herramientas de desarrollo como Visual Studio y Eclipse.
